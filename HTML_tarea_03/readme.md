## Tercera tarea HTML

En esta terca tarea trabajaremos sobre el archivo de la tarea anterior. 

## Consigna:

### HTML
- Agregar meta etiquetas a todas las paginas. (title, description, keywords.)
- Investigar otras metaetiquetas e incorporarlas si es posible.
- Armar una sección que contemple accesibilidad para personas con discapacidad
visual.
- Armar una sección que contemple accesibilidad para personas con discapacidad
auditiva.
- Utilizar elementos interactivos para que el sitio pueda recorrerse usando la
tecla TAB del teclado.

### CSS
- Crear una hoja global en CSS. (global.css)
- Crear una hoja para cada página. (ejemplo: contacto.html -> contacto.css)
- Elegir una paleta de colores de al menos dos colores ademas de blanco y negro.
- Colocar fondo de color a algunas seccion. La idea es usar esos fondos para
diferenciar una sección de otra.
- Investigar como colocar fuentes de texto externas (es decir que no vengan
ya con CSS). Aplicar una fuente generica a los textos y una fuente especial
a titulos y botones acorde a la tematica del sitio. Por ejemplo, si es un
sitio de video juegos se puede usar una fuente pixelada.
- Ajustar tamaños de titulos para que sea legible para personas medias chicatas.
- Poner en practica los diferentes selectores que vimos para practicar.
